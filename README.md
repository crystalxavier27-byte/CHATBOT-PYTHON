# CloudOps Chatbot 

This project is a DevOps chatbot built using Python, Flask, and Groq AI.

## Features

* Answers DevOps related questions
* Covers CI/CD, Docker, Kubernetes, Cloud
* Uses Groq Llama 3 AI model
* Web interface built with Flask

## How to Run

1. Install Python
2. Install required libraries

pip install flask groq

3. Get a Groq API key from
   https://console.groq.com/keys

4. Open `app.py` and replace

GROQ_API_KEY = "YOUR_GROQ_API_KEY" with your key.

5. Run the project
python app.py

6. Open browser
http://127.0.0.1:5000
